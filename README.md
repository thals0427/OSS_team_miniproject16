# OSS_team_miniproject16

## mini project 주제 

카카오톡 연락처 

## mini project 기능 

1. 친구 추가하는 기능
    - 이름
    - 생일 
    - 카카오 스토리 연동 유무
    - 즐겨찾기 유무 
    - 전화번호
    - 프로필 뮤직 
2. 친구 목록을 보는 기능
3. 친구 정보 수정하는 기능
4. 친구 삭제하는 기능
5. 친구 목록을 파일로 저장하는 기능
6. 친구 목록 파일을 불러오는 기능
7. 이름으로 친구를 검색하는 기능
    - 성 제외 검색
    - 초성 검색 가능
8. 전화번호로 친구를 검색하는 기능
    - 010 제외 검색 가능
    - 4자리 단위로 검색 가능
9. 생일으로 친구를 검색하는 기능
    - 年 검색 가능
    - 月 검색 가능
    - 日 검색 가능
